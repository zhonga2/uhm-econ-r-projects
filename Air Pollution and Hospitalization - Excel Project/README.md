# Air Pollution and Hospitalization — Excel Project

An October data analysis and visualization project completed for a business and economics spreadsheet modeling course. The study examines the relationship between **air pollutants and hospital emergency visits** across counties in **Massachusetts** and **Minnesota** (2015–2021), leveraging their comprehensive datasets.

## Files Included
- **(2015–2021) Massachusetts and Minnesota AQI.xlsx** — Air quality index data and pollutant trends.
- **Massachusetts-hospitalization-Chart.xlsx** — Data visualization of hospital visits correlated with AQI.
- **Anthony Zhong - Asthma Project.pptx** — Presentation summarizing findings.
- **Inx Box Table.pdf** — Supporting statistical tables and reference charts.

## Tools & Skills
Excel, data visualization, regression modeling, and quantitative analysis for decision-making in economics.
